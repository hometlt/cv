Студенческая работа – Разработка игрового движка на Java (2011/11 — 2011/12) https://sourceforge.net/projects/bresenham/ Работа, написанная в рамках курса изучения языка Java В программе использовались следующие технологии: • Box2D (физический движок, гравитация, столкновения объектов), • JOGL (Java OpenGL, графическая библиотека) • Jung (Java Universal Graph Framework, алгоритмы расчета кратчайшего пути для ИИ) • DOM4J (open source library for working with XML, сохранение и загрузка игры в форматах SVG и XML) • JMF (Java Media Framework API,поддержка звуковых эффектов) • Java Sockets (поддержка сетевой игры) • Android SDK (портирование на Android) • Работа как в виде веб-апплета, так и в виде приложения. • Реализация игр “Танки”, “Арканоид”, головоломок, основанных на взаимодействии гравитации, порталов и отражателей, визуализация генетических алгоритмов.
