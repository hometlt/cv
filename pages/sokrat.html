HTML5 Тренажер по сзематизации. (2012/09 – 2012/12) Мой проект для GamesInEdu. Разработка веб-тренажера по схематизации. Приложение используется как образовательный инструмент в школах менеджмента, таких как РАНХиГС и Тольяттинская Академия Управления. Программа дает задания студентам, затем студенты должны построить соответствующие данному заданию схемы. Программа включает в себя: • Графический движок, визуализатор схем, позволяющий создавать и редактировать схемы на веб-сайтах. Используются технологии Canvas и KineticJS. (В первой версии прграммы использовался SVG и RaphaelJS) • Анализатор схем. Бибилиотека, анализирующая построенные схемы. Позволяет давать оценку построенным схемам. Создан специальный интерпретируемый язык, позволяющий описывать правила проверки схем любой сложности. • Редактор задач. Программа работает во всех современных браузерах и iPad. My another Project at GamesInEdu. Development of Trainer of Schematization web-application. Application using as educational tool in Russian management Schools. Program giving tasks to students and then students have to build correct schemes depend on given tasks. Application includes: - Scheme render. JavaScript library that allow users to create schemes in the web - Scheme analyzer. Program analyzing created schemes and calculate the grade by specific rules. Rules written on specific interpretable scripting language. - Application using Object-oriented JavaScript and works in every modern desktop and mobile browsers. Technologies : HTML5, canvas/KineticJS (svg/RaphaelJS in old version)
